# SellingElectronics Backend

## Tech stack
Node.js, Express.js, TypeScript, MongoDB, Mongoose, JWT

## Features
- Authentication
- Product management
- Cart
- Order placement
- Order status tracking
- Error handling
- Controller-Service-Repository architecture

## How to run
npm install
npm run dev

## API endpoints
POST /api/auth/login
GET /api/products
POST /api/cart/items
POST /api/orders
...
